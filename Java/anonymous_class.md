# 익명 클래스(anonymous class)

## 정의

클래스의 선언과 객체의 생성을 동시에 하기 때문에 단 한번만 사용될 수 있고 오직 하나의 객체만을 생성할 수 있는 일회용 클래스
```text
new 조상클래스이름() {
	// 멤버 선언
}
또는
new 구현인터페이스이름() {
	// 멤버 선언ㅁ
}
```

<br/>

## 특징
- “익명” 이라는 단어 뜻대로 이름이 없기 때문에 생성자를 가질 수 없다
- 조상클래스의 이름 또는 구현하고자 하는 인터페이스의 이름을 사용해서 정의된다
- 정의할 때 사용한 조상클래스 하나를 상속받거나, 정의할 때 사용한 인터페이스 하나만 구현할 수 있다

<br/>

## 기타
- 이름이 없기 때문에 <b>외부 클래스명$숫자.class</b> 형식으로 클래스명이 결정된다
